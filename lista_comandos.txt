LISTA DE COMANDOS

1. git init : inicializa un repositorio en Git. Se habilita el seguimiento de versiones en este directorio.
2. git status : obtener informacion sobre el estado actual del repositorio.
3. git add : agregar nuevos archivos y/o directorios a un espacio temporal.
4. git commit -m "<mensaje>" : confirmar los cambios realizados en el espacio temporal.
5. git log : muestra un historial de los commits que se realizaron anteriormente.
6. git remote add origin <URL_repositorio_remoto> : conectar un repositorio local a un repositorio remoto.
7. git branch -M <nuevo_nombre> : cambiar el nombre de la rama principal (defecto master).
8. git push -u origin main // se utiliza para enviar cambios locales al repositorio remoto, y establecer una relacion de seguimiento entra la rama local "main" y la rama remota "origin".
